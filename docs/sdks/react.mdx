The Grat React SDK provides a set of hooks and providers to make it easy to integrate gas abstraction into your React frontend. It handles state management, loading indicators, and error handling out of the box.

## Installation

The React SDK requires `@grat-official-sdk/sdk` as a peer dependency.

<CodeGroup>
```bash npm
npm install @grat/react @grat-official-sdk/sdk
```

```bash yarn
yarn add @grat/react @grat-official-sdk/sdk
```

```bash pnpm
pnpm add @grat/react @grat-official-sdk/sdk
```
</CodeGroup>

## Setup

Wrap your application with the `GratProvider` to provide the configuration to all child components.

<CodeGroup>
```tsx Next.js (App Router)
// app/layout.tsx
import { GratProvider } from '@grat/react';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <GratProvider 
          apiKey={process.env.NEXT_PUBLIC_GRAT_API_KEY}
          network="testnet"
          relayUrl="https://api.grat.network"
        >
          {children}
        </GratProvider>
      </body>
    </html>
  );
}
```

```tsx Plain React
// App.tsx
import { GratProvider } from '@grat/react';

function App() {
  return (
    <GratProvider 
      apiKey="your_api_key"
      network="testnet"
      relayUrl="https://api.grat.network"
    >
      <MyApplication />
    </GratProvider>
  );
}
```
</CodeGroup>

## Hook Reference

### `useSponsor()`

Returns functions to sponsor transactions and manage the submission state.

```tsx
const { sponsor, sponsorContract, isLoading, error, reset } = useSponsor();
```

**Example: Send Payment Button**

```tsx
import { useSponsor } from '@grat/react';
import { TransactionBuilder, Asset, Operation, Networks } from '@stellar/stellar-sdk';

function SendPaymentButton({ user, destination, amount }) {
  const { sponsor, isLoading, error } = useSponsor();

  const handleSend = async () => {
    const tx = new TransactionBuilder(user, { fee: '100', networkPassphrase: Networks.TESTNET })
      .addOperation(Operation.payment({ destination, asset: Asset.native(), amount }))
      .setTimeout(30)
      .build();

    tx.sign(user);
    await sponsor(tx);
    alert('Payment sent successfully!');
  };

  return (
    <div>
      <button onClick={handleSend} disabled={isLoading}>
        {isLoading ? 'Sending...' : 'Send Sponsored Payment'}
      </button>
      {error && <p style={{ color: 'red' }}>Error: {error.message}</p>}
    </div>
  );
}
```

### `useSimulate()`

Simulates a Soroban transaction to return resource costs.

```tsx
const { simulate, isLoading, result, error } = useSimulate();
```

### `useEstimate()`

Provides a fee estimate for a transaction before submission.

```tsx
const { estimate, isLoading, result, error } = useEstimate();
```

### `useRelayStatus()`

Monitors the health of the connected Grat Relay. Auto-polls every 30 seconds.

```tsx
const { status, isHealthy, isLoading, error } = useRelayStatus();
```

### `useCreditBalance()`

Returns the current credit balance of your API key. Auto-refreshes on window focus.

```tsx
const { balance, totalDeposited, totalSpent, isLoading, error, refetch } = useCreditBalance();
```

### `useGrat()`

Returns the raw `Grat` client instance for advanced use cases not covered by hooks.

```tsx
const grat = useGrat();
```

## Full Example: Sponsor USDC Payment Form

This component demonstrates a complete flow including fee estimation and sponsorship.

```tsx
import React, { useState, useEffect } from 'react';
import { useSponsor, useEstimate } from '@grat/react';
import { TransactionBuilder, Asset, Operation, Networks } from '@stellar/stellar-sdk';

export function SponsorUSDCPayment({ userKeypair, usdcAsset }) {
  const [destination, setDestination] = useState('');
  const [amount, setAmount] = useState('');
  const { sponsor, isLoading: isSending, error: sendError } = useSponsor();
  const { estimate, result: feeEstimate, isLoading: isEstimating } = useEstimate();

  useEffect(() => {
    if (destination && amount) {
      const tx = new TransactionBuilder(userKeypair, { fee: '100', networkPassphrase: Networks.TESTNET })
        .addOperation(Operation.payment({ destination, asset: usdcAsset, amount }))
        .setTimeout(30)
        .build();
      estimate(tx);
    }
  }, [destination, amount]);

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    const tx = new TransactionBuilder(userKeypair, { fee: '100', networkPassphrase: Networks.TESTNET })
      .addOperation(Operation.payment({ destination, asset: usdcAsset, amount }))
      .setTimeout(30)
      .build();
    
    tx.sign(userKeypair);
    await sponsor(tx);
    alert('USDC Sent!');
  };

  return (
    <form onSubmit={handleSubmit} className="p-4 border rounded shadow">
      <h3>Sponsor USDC Payment</h3>
      <input 
        placeholder="Recipient G..." 
        value={destination} 
        onChange={e => setDestination(e.target.value)}
        className="w-full mb-2 p-2 border"
      />
      <input 
        placeholder="Amount" 
        value={amount} 
        onChange={e => setAmount(e.target.value)}
        className="w-full mb-2 p-2 border"
      />
      
      {feeEstimate && (
        <div className="text-sm text-gray-600 mb-2">
          Estimated Fee: {feeEstimate.estimatedFee} XLM (Sponsored)
        </div>
      )}

      <button type="submit" disabled={isSending || isEstimating} className="w-full bg-blue-600 text-white p-2 rounded">
        {isSending ? 'Processing...' : 'Send USDC'}
      </button>

      {sendError && <p className="text-red-500 mt-2">{sendError.message}</p>}
    </form>
  );
}
```

## Error Handling

The hooks expose a typed `error` object. You can catch specific Grat errors for more granular UI feedback.

```tsx
import { AuthenticationError, RateLimitError } from '@grat-official-sdk/sdk';

// Inside a component
const { sponsor, error } = useSponsor();

if (error instanceof AuthenticationError) {
  return <p>Invalid API Key. Please check your configuration.</p>;
}

if (error instanceof RateLimitError) {
  return <p>Too many requests. Please try again in {error.retryAfter} seconds.</p>;
}
```

<Tip>
  Always use `useEstimate()` before sponsoring a transaction to provide transparency to your users about the fees being covered.
</Tip>

<Note>
  `@grat-official-sdk/sdk` is a required peer dependency and must be installed alongside the React SDK.
</Note>
