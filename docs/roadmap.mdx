Grat is being built in the open. Here's what's available today and what's coming. We ship in phases — each one adds real, usable functionality.

<Tip>
  Everything on this roadmap is being built in the open. Watch the GitHub repo to follow along.
</Tip>

## Available Now

<CardGroup cols={2}>
  <Card title="Core Relay Server" icon="circle-check" color="#22c55e">
    Self-host via Docker to sponsor Stellar and Soroban transactions on the testnet.
  </Card>
  <Card title="TypeScript SDK" icon="circle-check" color="#22c55e">
    Install from npm and integrate with your project in under 10 lines of code.
  </Card>
  <Card title="Soroban Support" icon="circle-check" color="#22c55e">
    Simulate contract calls, estimate resource fees, and sponsor invocations natively.
  </Card>
  <Card title="Docker Self-Hosting" icon="circle-check" color="#22c55e">
    One-command deployment with docker-compose and Redis-backed channel management.
  </Card>
  <Card title="Fee Channel Manager" icon="circle-check" color="#22c55e">
    HD-derived channel pool with locking, auto-release, and round-robin selection.
  </Card>
</CardGroup>

## In Progress

<CardGroup cols={2}>
  <Card title="Mainnet Deployment" icon="hammer" color="#f59e0b">
    Sponsor real transactions with funded channel accounts on the Stellar mainnet.
  </Card>
  <Card title="API Key Authentication" icon="hammer" color="#f59e0b">
    Scoped keys for mainnet and testnet with support for rotation and revocation.
  </Card>
  <Card title="Credit System" icon="hammer" color="#f59e0b">
    Deposit XLM, track your balance, and enable automatic fee deduction per transaction.
  </Card>
  <Card title="Rate Limiting" icon="hammer" color="#f59e0b">
    Per-key and per-account limits implemented with Redis-backed sliding windows.
  </Card>
  <Card title="Launchtube Migration" icon="hammer" color="#f59e0b">
    Drop-in replacement with a two-line code change for projects moving from Launchtube.
  </Card>
</CardGroup>
