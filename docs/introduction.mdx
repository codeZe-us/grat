Grat is a relay server and SDK that implements **Gas Abstraction** for the Stellar blockchain. It allows developers to sponsor transaction fees for their users, enabling a seamless UX where users never need to hold or acquire XLM.

Grat supports all Stellar transaction types including **Protocol 26 (Yardstick)** operations: checked integer arithmetic, BN254 cryptographic functions, precise TTL extensions, and SAC improvements.

## How it works

<Steps>
  <Step title="Sign">
    The user signs a standard transaction (the "inner" transaction) as they normally would.
  </Step>
  <Step title="Relay">The SDK sends the signed XDR to the Grat Relay instead of Horizon.</Step>
  <Step title="Sponsor">
    The Relay wraps the transaction in a **Fee-Bump envelope**, pays the network fee from a channel
    pool, and submits it to Stellar.
  </Step>
</Steps>
