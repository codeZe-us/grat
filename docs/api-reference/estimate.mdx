---
title: Estimate Fee
api: POST /v1/estimate
description: "Predict the total cost of a transaction without submitting it."
---

Use this endpoint to provide users with a "gas estimate" before they sign a transaction. It combines Horizon fee stats and Soroban simulation (if applicable) to give a complete cost breakdown.

### Request Body
<ParamField body="transaction" type="string" required>
  The Base64-encoded XDR of the transaction.
</ParamField>

### Response
<ResponseField name="estimatedFee" type="string">
  The total suggested fee in stroops.
</ResponseField>

<ResponseField name="breakdown" type="object">
  <Expandable title="properties">
    <ResponseField name="baseFee" type="string">
      The network inclusion fee.
    </ResponseField>
    <ResponseField name="resourceFee" type="string">
      The Soroban resource fee (if Soroban).
    </ResponseField>
  </Expandable>
</ResponseField>

### SDK Example
```typescript
const estimate = await grat.estimate(tx);
console.log(`Expected cost: ${estimate.estimatedFee} stroops`);
```
